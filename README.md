# RAIKU Simulator

Active HTML simulator UI for the RAIKU revenue model.

## Current entrypoint

- `index.html`

## Data contract

- AOT simulator data is loaded from `data/aot_programs.v1.js`
- Upstream source-of-truth for taxonomy, pipeline outputs, and artifact generation lives in the separate `raiku-revenue-model` repo
